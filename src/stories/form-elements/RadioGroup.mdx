# RadioGroup

A RadioGroup represents a group of radio buttons which can be selected for different options. The radio buttons are grouped together and only one option can be selected at a time.
Selecting can be done by clicking on the radio button or by using the arrow keys of the keyboard. 

[API Reference](https://www.radix-ui.com/primitives/docs/components/radio-group)
